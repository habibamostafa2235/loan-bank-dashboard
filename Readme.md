📊 Loan Bank Dashboard

📌 Project Overview
This project is an interactive Excel-based dashboard designed to analyze loan data and provide meaningful insights into loan approvals, repayment behavior, and borrower risk profiles. The dashboard helps in understanding key financial patterns and supports data-driven decision-making.

🎯 Objectives
Analyze loan approval and rejection trends
Identify borrower risk categories
Track repayment behavior and default rates
Provide clear and interactive visual insights

🛠️ Tools & Techniques Used
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning & Transformation
Data Visualization

📈 Key Features
Interactive dashboard for loan performance analysis
Categorization of borrowers based on risk levels
KPIs such as total loans, default rate, and approval rate
Clear visual representation of financial data

💡 Outcome
This project improved my skills in data analysis, dashboard creation, and data visualization using Excel, and enhanced my ability to extract insights from raw financial data.
